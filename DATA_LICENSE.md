# Data License

Self-created metadata, dataset statistics, normalized label taxonomy, schema documentation, and annotation mappings in this repository are released under the Creative Commons Attribution 4.0 International License (CC BY 4.0), subject to compatibility with upstream source terms.

Raw upstream datasets are not redistributed in this repository.

Processed benchmark samples that contain upstream source code are released only where the corresponding upstream license permits redistribution. For sources with unclear or mixed redistribution terms, this repository provides source pointers, retrieval scripts, derived metadata, and reconstruction instructions rather than raw upstream archives.

Users must also comply with the licenses and citation requirements of the original upstream sources.
