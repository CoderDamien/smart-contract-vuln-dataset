# Data License Draft

Self-created metadata, dataset statistics, normalized label taxonomy, schema documentation, and annotation mappings in this repository are intended to be released under the Creative Commons Attribution 4.0 International License (CC BY 4.0).

Raw upstream datasets are not redistributed in this repository.

Processed benchmark samples that contain upstream source code should only be published when the corresponding upstream license permits redistribution. Otherwise, the release should provide source pointers, retrieval scripts, and reconstruction instructions.

Users must also comply with the licenses and citation requirements of the original upstream sources.
